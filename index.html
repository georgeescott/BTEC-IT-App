Course Structure

If you want to prepare for work and study in the fast-moving technology sector or are keen to develop strong IT knowledge and skills that are valued in any industry, then a BTEC in IT is the right choice for you.

BTEC courses are vocational qualifications that provide an alternative method in which to gain the essential skills and knowledge for future study or a career in Information Technology. They offer you the chance to gain a nationally recognised qualification on a more practical level in the way that you are both taught and assessed. BTEC IT courses we offer:

Extended Diploma in IT (Software Development)

Broadly equivalent to 3 A Levels. This is a two year course that is typically chosen by students planning to enter a career in IT or study Computing and IT at university.

Students on the Extended Diploma also currently have an opportunity to take a professional exam to become a Microsoft Technology Associate.

Subsidiary Diploma in IT

Broadly equivalent to 1 A Level. In the first year you will work towards completing a BTEC National Level 3 Certificate in Information Technology.

Should you wish to continue with this qualification in the second year you will then work towards completing a BTEC National Level 3 Subsidiary Diploma in Information Technology.

What will you learn?

Some typical Subsidiary Diploma units include:



Computer Systems:

This unit will involve practical work installing, configuring and testing hardware and software.



IT Technical Support:

Learners will look at tools for IT support and communicating advice; work with organisational policies; and study techniques for gathering information on faults.



Website Production:

There are over 250 million websites on the Internet and the competition to get noticed is high. Learners will design websites and explore factors that influence performance and capture interest.



Human Computer Interaction:

How people interact with technology such as PCs, mobile phones and television remotes. This unit explores how HCI has developed and where it might be going next, with particular emphasis on website and software interface design.



Digital Graphics:

In this unit learners will create and select suitable images for interactive media and explore file formats. Learners will develop skills with the Adobe Creative Suite.



Communication and Employability Skills:

Non-technical skills, known as soft skills, are required alongside technical skills in an IT industry full of jargon and it is particularly important that learners are able to communicate with non-technical staff.



Web Animation for Interactive Media:

Learners will develop practical skills in the creation of interactive animations designed for websites




Some typical Extended Diploma (Software Development) units include:



Software Design and Development:

Programming using Visual Basic .NET, introducing concepts such as iteration and selection.



Event Driven Programming:

Programming Windows applications with graphical user interfaces using Visual Basic .NET.



JavaScript and CSS:

Programming interactive web pages with JavaScript and working with the CSS box model for advanced layout of web pages.



Developing Computer Games:

In addition to researching the impact of games on society, students will extend their programming skills from earlier units, looking at topics such as collision detection, difficulty levels and random behaviour.



Spreadsheet Modelling:

Developing and testing automated spreadsheets to solve complex problems.



Database Design:

Designing and creating relational databases - the technology behind the data handling solutions of most of the large websites you use such as Twitter, Wikipedia and Facebook.



Information Systems:

How companies use and store data to give them a competitive advantage.

Frequently Asked Questions:


Can I study BTEC IT without a GCSE in IT?

Yes, the course assumes no prior knowledge.



How is the course assessed?

There are no exams, only coursework. Assignments can be in written form, but it is just as likely they will be in other forms such as videos, presentations, website designs, databases, podcasts, programming, or blog.



Can I take BTEC IT and Computing A level at the same time?

No. Even though they are different courses, they share many topics and taking both would not be effective use of your time.



Are BTECs recognised by universities?

Yes. Learners who gain a BTEC Level 3 National qualification will earn UCAS points at the same time and this qualification gives UCAS points equivalent to one AS level, one A level or three A levels, depending on pathway taken.



Will I have to study other subjects as well as a BTEC?

It depends: If you are choosing to study a 1 A level equivalent qualification (Certificate and/or a Subsidiary Diploma) then yes; these qualifications will make up one of your subject choices alongside your A Levels. If you are choosing the 3 A Level equivalent qualification (Extended Diploma) then no; however, you might want to think about an additional AS Level that might strengthen your academic profile and complement your BTEC qualification.

Example 1 - HTML

HTML (HyperText Markup Language) is the computer language used for the content you see on a web page. It consists of code called tags. There are many HTML tags, some examples would be: bold text, paragraphs, headings, sections and footers. Tags look like this:


<strong></strong> (bold)
<p></p> (paragraph)
<em></em> (italics)
<h1></h1> (heading)
<u></u> (underline)

To begin an HTML tag, you need to enclose it in < > symbols, with the tags name/letter inside. To end an HTML tag you do the same as opening it, but insert a / forward slash before the name of the tag. Any content you want, goes inbetween the two starting and ending tags.

Example 4 - JavaScript

Now that you've completed the first Javascript example, you can go onto something harder! Below is an example of a For Loop. A for loop is very useful and time saving. If you wanted to write the 5 times table using javascript you might try and code it like this:


document.write(5*1 + "<br>");
document.write(5*2 + "<br>");
document.write(5*3 + "<br>");
document.write(5*4 + "<br>");
document.write(5*5 + "<br>");
document.write(5*6 + "<br>");

But that takes a long time, and will make your code very long. Instead, you can use a for loop to loop through the 5 times table, adding 1 to the number 5 is being multiplied by. For example:

for (var i=1; i<=10; i++){
    document.write("5 x " + i + " = " + 5*i + "<br/>");
}

The word "var" simply means "new variable", somewhere where we are telling the computer to store a value.

In this case, the variable "i". which has been set to 1 so that the loop will start with 1x5. "i<=10" means that the loop should stop at 10x5, and i++ means add one every time it loops.

Example 5 - CSS

CSS (cascading style sheet) is what styles a web page. You can change the appearance, layout and size of a web page. With CSS, you have to first select the html tag that you want to style. You then use two { } curly brackets to put the styles in. Below is an example of how you'd change the color of text on a web page:


html {color: red;}

When using colours in CSS, you can use the name of the colour like above, or a hex colour code, which is made up of 6 digits; for example, white in hex is #FFFFFF and black is #000000. Have a go below at changing the color of the text!
